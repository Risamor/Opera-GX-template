- 🎬 **Video de fondo animado** - Animación en la pantalla de inicio (Tambien hay otro codigo para imagenes estaticas)
- 🎵 **Música** - Banda sonora personalizada
- 🎨 **Paleta de colores** - (Los colores que usted elija) Ejmplo: Rojo (#FF0033) y morado (#660066)
- 🌙 **Modo oscuro/claro** - Compatible con ambos temas

## 📥 Instalación

1. Comprime el archivo `Template.zip`
2. Ve a `https://dev.gx.games/mods`
3. Haga Click en `Upload new mod`
4. Busca tu archivo `.zip` con choose file
5. Y presionar `Confirm`
